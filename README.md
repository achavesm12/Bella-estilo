# Bella Estilo

Bella Estilo es una **aplicación web de comercio electrónico** para la venta de ropa femenina, desarrollada con **ASP.NET Core MVC**, que permite a los usuarios navegar productos, agregar artículos al carrito, realizar el checkout y recibir su factura en PDF por correo electrónico.

Este proyecto fue desarrollado como parte de mi formación en Ingeniería de Software y representa un sistema completo con arquitectura en capas, buenas prácticas y funcionalidades reales de un e-commerce.

---

## Funcionalidades principales

- Catálogo de productos con categorías, etiquetas e imágenes múltiples
- Vista de detalle del producto con reseñas y valoraciones
- Carrito de compras con productos normales y personalizados
- Actualización dinámica de cantidades
- Proceso de checkout con validaciones de tarjeta
- Generación de factura en **PDF**
- Envío automático de factura por **correo electrónico**
- Gestión de promociones y descuentos
- Arquitectura en capas (Application, Infrastructure, Web)

---

## Arquitectura del proyecto

El proyecto está organizado siguiendo una arquitectura en capas:

- **BellaEstilo.Application** → Lógica de negocio y servicios  
- **BellaEstilo.Infrastructure** → Acceso a datos, repositorios y contexto  
- **BellaEstilo.Web** → Capa web (MVC, vistas y controladores)


Esto permite un código más mantenible, escalable y fácil de probar.
